# Задание 01 (Процессы)

## Описание

Программа принимает аргументы командной строки и обрабатывает их по следующим правилам:

1. если аргумент является целым или вещественным числом, выводится само значение и значение, умноженное на 2;
2. если аргумент не является числом, он выводится в исходном виде.

Обработка аргументов выполняется двумя процессами:
- родительским;
- дочерним.

Аргументы делятся между ними примерно поровну.

---

## Используемые средства

В программе используются:
- аргументы командной строки `argc`, `argv`;
- системный вызов `fork()` для создания дочернего процесса;
- `wait()` / `waitpid()` для ожидания завершения дочернего процесса;
- функция `strtod()` для проверки, является ли строка числом.
