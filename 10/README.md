# Задание 10. Разделяемая память System V

## Описание задания

Родительский процесс генерирует наборы из случайного количества случайных чисел и помещает их в разделяемую память.  
Дочерний процесс считывает эти числа из разделяемой памяти, находит минимальное и максимальное значения, после чего записывает результат обратно в разделяемую память.  
Родительский процесс считывает найденные минимум и максимум и выводит их на экран.  

Процесс повторяется до получения сигнала `SIGINT` (`Ctrl+C`), после чего программа завершает работу и выводит количество обработанных наборов данных.

---

## Используемые технологии

В программе используются:

- `fork()` — создание дочернего процесса
- `signal()` — обработка сигнала `SIGINT`
- `shmget()` — создание сегмента разделяемой памяти
- `shmat()` — подключение разделяемой памяти к процессу
- `shmdt()` — отключение разделяемой памяти
- `shmctl()` — удаление сегмента разделяемой памяти
- `wait()` — ожидание завершения дочернего процесса

---

## Логика работы программы

Программа состоит из двух процессов:

### Родительский процесс
- создаёт сегмент разделяемой памяти;
- генерирует случайный набор чисел;
- записывает его в shared memory;
- ждёт, пока дочерний процесс вычислит минимум и максимум;
- выводит результат на экран;
- считает количество обработанных наборов.

### Дочерний процесс
- ждёт появления нового набора чисел в shared memory;
- считывает числа;
- находит минимальное и максимальное значения;
- записывает результат обратно в shared memory.

---

## Синхронизация процессов

Для согласования работы родительского и дочернего процессов используется поле состояния `state` в общей структуре.

Возможные состояния:

- `STATE_EMPTY` — память свободна, можно записывать новый набор;
- `STATE_DATA_READY` — родитель записал данные, дочерний процесс может обрабатывать;
- `STATE_RESULT_READY` — дочерний процесс вычислил минимум и максимум;
- `STATE_FINISH` — завершение программы.

---

## Структура данных в разделяемой памяти

В shared memory хранится структура:

- количество чисел в текущем наборе;
- массив чисел;
- минимальное значение;
- максимальное значение;
- текущее состояние обмена.
