# UDP Chat for Two Clients

## Описание
Данная программа реализует простой чат для двух клиентов с использованием UDP-сокетов.  
Каждый экземпляр программы запускается в отдельном терминале, привязывается к своему локальному порту и обменивается сообщениями с другим экземпляром программы по IP-адресу и порту.

Асинхронное взаимодействие реализовано с помощью `fork()`:
- дочерний процесс постоянно ожидает входящие сообщения через `recvfrom()`;
- родительский процесс считывает ввод пользователя и отправляет сообщения через `sendto()`.

## Особенности реализации
- используется протокол UDP (`SOCK_DGRAM`);
- каждый клиент знает:
  - свой локальный порт;
  - IP-адрес собеседника;
  - порт собеседника;
- один процесс отвечает за приём сообщений;
- второй процесс отвечает за отправку сообщений;
- при завершении отправляющего процесса дочерний процесс также корректно завершается.

## Используемые функции
- `socket()` — создание UDP-сокета;
- `bind()` — привязка сокета к локальному порту;
- `sendto()` — отправка сообщения собеседнику;
- `recvfrom()` — получение сообщения;
- `fork()` — создание дочернего процесса для параллельной работы;
- `kill()` — завершение дочернего процесса;
- `wait()` — ожидание завершения дочернего процесса;
- `close()` — закрытие сокета.
