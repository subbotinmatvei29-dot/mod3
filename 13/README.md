# TCP Client-Server 13

## Описание

Программа состоит из двух частей:

- **server** — TCP-сервер
- **client** — TCP-клиент

Сервер принимает подключения от клиентов по TCP и обрабатывает одну команду за одно соединение.

Поддерживаются два режима работы:

1. **Вычисления**
   - сложение
   - вычитание
   - умножение
   - деление

2. **Передача файла**
   - клиент отправляет файл серверу
   - сервер сохраняет файл у себя в текущей директории

---

## Поддерживаемые команды протокола

### 1. Арифметическая команда

Формат:
CALC <операция> <число1> <число2>

Примеры:
CALC + 10 5
CALC - 10 5
CALC * 10 5
CALC / 10 5

Ответы сервера:
- RESULT <значение>
- ERROR division by zero
- ERROR unknown operation
- ERROR invalid format

---

### 2. Передача файла

Формат заголовка:
FILE <имя_файла> <размер_в_байтах>

После заголовка клиент отправляет содержимое файла.

Ответы сервера:
- OK FILE RECEIVED
- ERROR invalid file header
- ERROR invalid file size
- ERROR cannot open output file
- ERROR receiving file data
- ERROR writing file
