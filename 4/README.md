# Задание 04. Каналы

Программа `shell` представляет собой простой командный интерпретатор.

Поддерживается:
- запуск внешних программ;
- конвейер через `|`;
- перенаправление ввода через `<`;
- перенаправление вывода через `>`.

Также в папке находятся программы:
- `concat`
- `sum`
- `maxlen`

Их можно запускать из интерпретатора.

## Сборка

```bash
make


ls
ls | sort
echo hello > out.txt
cat out.txt
cat < out.txt | wc
sum 1 2 3
concat hello world
maxlen abc abcd xy
