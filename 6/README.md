# Групповой чат на очередях сообщений System V

## Описание

Программа реализует групповой чат с использованием одной очереди сообщений System V.

Есть две программы:

- `server` — создает очередь сообщений, принимает сообщения от клиентов и пересылает их другим клиентам.
- `client` — подключается к очереди, отправляет сообщения серверу и получает сообщения, адресованные этому клиенту.

## Логика работы

- Для сервера используется тип сообщения `10`.
- Для клиентов используются типы `20`, `30`, `40`, ...
- Клиент отправляет сообщение серверу:
  - `mtype = 10`
  - `sender = номер клиента`
  - `text = текст сообщения`
- Сервер получает сообщение, запоминает клиента и пересылает текст всем активным клиентам, кроме отправителя.
- Если клиент отправил строку `shutdown`, сервер перестает пересылать сообщения этому клиенту.

## Структура файлов

- `chat.h` — общие константы и структура сообщения
- `server.c` — сервер
- `client.c` — клиент
- `Makefile` — сборка проекта
