# Задание 09. Семафоры POSIX

## Описание

Программа демонстрирует синхронизацию двух процессов с помощью **POSIX-семафора** при работе с общим файлом.

По условию задания:

- **родительский процесс** генерирует строки, состоящие из случайного количества случайных чисел, и записывает их в файл;
- **дочерний процесс** анализирует новые строки файла: для каждой строки находит **минимальное** и **максимальное** число и выводит результат на экран;
- доступ к файлу должен быть защищён **семафором**.

В программе используется **именованный POSIX-семафор**, который обеспечивает взаимоисключающий доступ к файлу: в каждый момент времени с файлом работает только один процесс.

---

## Логика работы программы

1. В начале работы очищается файл `data.txt`.
2. Создаётся именованный POSIX-семафор со значением `1`.
3. После вызова `fork()` появляются два процесса:
   - **родительский** процесс:
     - генерирует строку со случайными числами;
     - захватывает семафор;
     - записывает строку в конец файла;
     - освобождает семафор;
   - **дочерний** процесс:
     - захватывает семафор;
     - открывает файл на чтение;
     - читает только **новые**, ещё не обработанные строки;
     - для каждой строки вычисляет `min` и `max`;
     - освобождает семафор.
4. После завершения работы процессы закрывают семафор, а родитель удаляет его из системы.

---

## Используемые средства

- `fork()` — создание дочернего процесса
- `sem_open()` — создание/открытие именованного POSIX-семафора
- `sem_wait()` — захват семафора
- `sem_post()` — освобождение семафора
- `sem_close()` — закрытие семафора
- `sem_unlink()` — удаление именованного семафора
- `fopen()`, `fprintf()`, `fgets()`, `fseek()`, `ftell()` — работа с файлом
